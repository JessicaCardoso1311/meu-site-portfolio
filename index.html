Projeto 1: Site de Portfólio Pessoal (Implementação Web)

Este site é meu projeto de introdução à implementação web. O objetivo foi criar uma vitrine profissional para meus futuros projetos de Análise de Dados.

Utilizando ferramentas de IA como assistente para a geração da estrutura base de HTML/CSS, meu foco principal foi na **implementação, personalização e solução de problemas (debugging)** . O maior desafio foi o processo de *deploy* (implantação) no GitHub Pages, superando uma série de obstáculos técnicos com nomes de arquivos, cache do navegador e o tradutor automático para conseguir publicar o site com sucesso.

Ferramentas e Conceitos: HTML5, CSS3, Git, GitHub Pages, Deploy, Debugging e Assistência de IA.
