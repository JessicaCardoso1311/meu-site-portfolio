Projeto 1: Site de Portfólio Pessoal (HTML/CSS)

Este próprio site é meu primeiro projeto, desenvolvido do zero para servir como minha vitrine profissional. O objetivo foi criar uma página web responsiva e visualmente atraente para centralizar meus futuros projetos de Análise de Dados.

O desenvolvimento envolveu a aplicação de conceitos de HTML5 para a estrutura e CSS3 para a estilização, incluindo o efeito de fundo fixo (parallax) e caixas de conteúdo semi-transparentes. Todo o processo foi versionado com Git e hospedado no GitHub Pages.

Ferramentas Utilizadas: HTML5, CSS3, Git, GitHub e GitHub Pages.
